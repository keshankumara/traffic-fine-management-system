# Traffic Fine Management System

> A multi-platform traffic fine management solution for Sri Lanka Police, providing a Spring Boot backend, a React/Vite citizen portal, and a Flutter mobile app.

---

## ✨ Features

- Secure backend with Spring Security and JWT authentication
- Fine search and management system
- Citizen web portal (React + Vite)
- Flutter mobile application
- Admin dashboard and analytics
- REST API with OpenAPI/Swagger documentation
- PostgreSQL / H2 database support
